# Ecomerce Backend

This project build a website that sell electronic device. (B2C)

## Description

- Backend: Flask
- Database: Mysql

## Installation

Install Mysql Database

Install python

Install lib

```bash
$ pip install -r requirements.txt
```

## Running the app

Init database (only once at start)

```bash
$ ./migrate.sh
```

Run API
```bash
$ python main.py
```
